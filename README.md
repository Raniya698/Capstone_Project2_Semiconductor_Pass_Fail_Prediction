# Semiconductor Pass/Fail Prediction

## About the Project

This project was completed as part of my capstone work to develop a machine learning model for predicting pass or fail outcomes in semiconductor manufacturing. The dataset contains sensor measurements collected during the manufacturing process.

The project focuses on cleaning the dataset, handling missing values, preprocessing features, training multiple machine learning models, and comparing their performance.

## Tools and Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## Dataset

The semiconductor manufacturing dataset contains hundreds of sensor measurements used to determine whether a manufactured product passes or fails quality inspection.

## Project Workflow

- Imported the dataset
- Handled missing values
- Removed non-numeric features
- Standardized the data
- Split the dataset into training and testing sets
- Trained Random Forest, SVM, and Naive Bayes models
- Compared model performance
- Selected the best-performing model

## Results

Among the three machine learning models, Random Forest achieved the highest prediction accuracy of approximately **92.36%**, making it the best-performing model for this dataset.
